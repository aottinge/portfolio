Projet 2 : Création d'un projet Flask

Analyse : Ce projet m'a permis de découvrir le développement web côté serveur avec Python et le micro-framework Flask. J'ai conçu une application web dynamique avec gestion des routes, des templates et des formulaires.

Trace : Le projet inclut une authentification utilisateur, la gestion de sessions et l'affichage dynamique de données. J'ai également utilisé Jinja2 pour générer les pages HTML à partir de modèles.

Ce travail m'a donné une première expérience concrète du développement back-end et de l'intégration entre le front-end et le back-end.
